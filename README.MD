## Dependency

You need to add jitpack first

In maven:

``` xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependency>
    <groupId>com.github.coolclk</groupId>
    <artifactId>ConsoleCode</artifactId>
    <version>1.0</version>
</dependency>
```

In gradle:

``` groovy
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.coolclk:ConsoleCode:1.0'
}
```
